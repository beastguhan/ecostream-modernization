# EcoStream Infrastructure Modernization

This project modernizes EcoStream’s production infrastructure using AWS and Terraform.

Components:

- Dockerized Hello World application
- Infrastructure as Code using Terraform
- Amazon ECS for container orchestration
- Application Load Balancer for traffic distribution
- Amazon RDS with Multi-AZ for database resilience
- Secure networking using VPC with public and private subnets

Deployment steps:

terraform init
terraform plan
terraform apply
